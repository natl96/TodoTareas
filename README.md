# TodoTareas

App para gestionar tareas del día a día. Creada con React y TypeScript.

🔗 https://todotareas.onrender.com

## ¿Qué puedes hacer?

- Registrarte e iniciar sesión
- Crear tareas con título, descripción, fecha límite y prioridad
- Ver tus tareas organizadas en tres columnas: Pendiente, En Progreso y Completada
- Editar o eliminar tareas
- Buscar por texto o filtrar por estado y prioridad

## Tecnologías

- React 18 + TypeScript
- CSS3
- Desplegado en Render.com

## Cómo correrlo localmente

```bash
npm install
npm start
```
